# Index Olution

Site web basé sur PHP avec Bootstrap, Swiper et autres composants front-end.

## Prérequis

- PHP (recommandé : 7.4+)
- Serveur web (Apache, Nginx) ou `php -S localhost:8000` pour le développement

## Installation

1. Cloner le dépôt :
   ```bash
   git clone https://github.com/VOTRE_UTILISATEUR/index_olution.git
   cd index_olution
   ```

2. Lancer un serveur PHP local (optionnel) :
   ```bash
   php -S localhost:8000
   ```

3. Ouvrir http://localhost:8000 dans un navigateur.

## Structure

- `index.php` — Page principale
- `assets/` — CSS, JS, images et bibliothèques (Bootstrap, Swiper, etc.)
